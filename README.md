# ReLab Web

## Descripción
Aplicación web interactiva desarrollada con React que permite la visualización,
búsqueda y filtrado dinámico de productos, usuarios y categorías, consumiendo
una API propia desarrollada en Spring Boot.

## Tecnologías utilizadas
- React + TypeScript
- Vite
- React Router DOM
- React Hook Form
- API REST (Spring Boot)
- Cloudinary
- HTML5 / CSS3
- Git y GitHub

## Funcionalidades principales
- Consumo de datos desde múltiples endpoints de una API REST
- Navegación mediante React Router con rutas dinámicas
- Filtrado, búsqueda y ordenación en tiempo real
- Paginación de resultados
- Vista de detalle de productos
- Subida de imágenes usando Cloudinary
- Formulario con validaciones
- Tema claro / oscuro persistente mediante LocalStorage
- Diseño responsive con menú hamburguesa en dispositivos móviles

## Puesta en marcha del proyecto
```bash
npm install
npm run dev
```
